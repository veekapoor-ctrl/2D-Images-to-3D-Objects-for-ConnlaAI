# 2D-Images-to-3D-Objects-for-ConnlaAI
Methods used to convert 2D flat Images (JPEG,PNG) to 3D objects (STL,glb,glTF) for flagship product Renaissance Digital Library. 
The Renaissance Digital Library is a flagship product of the ConnlaAI, a cultural-heritage startup that houses 15th and 16th Century datasets ranging from digitised books, illustrations, and other visual features. Supported by Enterprise Ireland’s Commercialisation Scheme, the product is an interesting combination of digital tools and historical datasets, something that does not exist in the current historiographical sphere. 

Read the Step by Step guide for instructions
